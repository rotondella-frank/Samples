This repository is to sample Tableau, Python, and other technical skills.

Tableau: 
This folder contains a Tableau workbook as well as pdfs of the dashboards. To see drill downs and other dynamic features downloading the workbook is best. This covers sales, financial statistics, and demographics. 

VIX_arima_forecast_py:
This folder contains a forecast of the vix using the FRED API. This also includes a file with the graph of the output, with the red line showing the forecast. 

Vix_summary_stat_py:
This folder contains a set of summary statistics to infer that the bottom decile of the vix can be very profitable to purchase as it has a quick recovery and high percentage movements. This can be run as python code, or viewed through attached graphs as with other sample folders.
